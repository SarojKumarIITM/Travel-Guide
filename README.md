# Travel Guide Website (HTML Project)

This is a simple Travel Guide website built using only HTML.  
The project showcases popular travel destinations in India with a clean and user-friendly layout.

## 🚀 Features
- Displays multiple travel destinations
- Simple and clean user interface
- Easy navigation using basic HTML structure
- Beginner-friendly project

## 🛠️ Technologies Used
- HTML5

This project was created to practice and demonstrate basic HTML skills such as:
- Page structure
- Semantic tags
- Content organization
